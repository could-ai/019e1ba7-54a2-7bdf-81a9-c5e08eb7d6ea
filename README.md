# Smart Tasty 🥗

Smart Tasty est une application mobile de commande de repas sains (poke bowls, salades de pâtes, plats végétariens) conçue dans un style moderne (vert et blanc) rappelant Uber Eats.

## Fonctionnalités

- **Catalogue de Repas Healthy** : Parcourez une sélection de repas froids et sains.
- **Détails et Nutrition** : Consultez les ingrédients et les valeurs nutritionnelles de chaque plat.
- **Commande Avancée** : Commandez vos repas pour le lendemain ou pour plusieurs jours à l'avance.
- **Option Éco-responsable** : Ajoutez des couverts en bois pour 10 roupies.
- **Panier et Paiement** : Payez avec le solde de votre portefeuille intégré.
- **QR Code** : Recevez un QR code pour récupérer votre repas en toute simplicité.
- **Gestion de Profil** : Suivez le solde de votre compte et annulez facilement vos commandes si vous changez d'avis.

## Technologies Utilisées

- [Flutter](https://flutter.dev) (iOS, Android, Web, Desktop)
- Provider (Gestion d'état)
- qr_flutter (Génération de QR codes)

## Lancement Rapide

1. Assurez-vous d'avoir Flutter installé.
2. Clonez ce dépôt.
3. Exécutez `flutter pub get` pour installer les dépendances (notamment `provider` et `qr_flutter`).
4. Lancez l'application avec `flutter run`.
